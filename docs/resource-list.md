- **Install Anaconda**
- **Install Jupyter Lab**
- **Set up Github Repo**
    - Create a repo & clone it
    - Add README.md
    - Push to main
    - Set credentials (use a Personal Access Token from Github account)
    - Install & enable JupyterLab Git Extension
